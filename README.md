# SOEN 287 Course Management System

1. Project Description
This website is a web-based Smart Course Companion that helps students organize their courses, manage assessments, track grades, and visualize their academic progress.

2. Features Implemented

    General Features:
    - Log in Page
    - Dashboard

    Student Features:
    - View Courses
    - View Grades
    - Track Grade Progress

    Teacher Features:
    - View Students
    - Manage Grades

3. Installation Steps

    a) Download the project files
    b) Extract the compressed folder
